## Installation project

##### 1. Clone this repository on your machine.
##### 2. Create your environment configuration file from example:
```
$ cp env.example .env
```
##### 3. Set config data to `.env` file.
##### 4. Run the following commands:
```
composer install
php artisan app:install
```
